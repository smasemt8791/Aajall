
آجال – Aajall PWA  
تطبيق المحامين الذكي لإدارة المواعيد والآجال القانونية بعداد تنازلي دقيق وتنبيهات ذكية

#### نبذة عن المشروع
«آجال» هو تطبيق ويب تقدمي (Progressive Web App) احترافي مصمم خصيصًا للمحامين ومكاتب المحاماة والمستشارين القانونيين. يس
يُعدّ أداة لا غنى عنها لتتبع الآجال القانونية الحرجة، مواعيد الجلسات، ومهل الطعن والاستئناف بدقة متناهية، مع ضمان عدم تفويت أي موعد مصيري مهما كان حجم القضايا.

تم تطوير التطبيق بالكامل بواسطة أخصائي معلوماتية صحية وقانونية ومطور Full-Stack، مع التركيز على الأداء العالي، العمل بدون إنترنت، والأمان التام لبيانات القضايا.

#### الميزات الرئيسية
- عدّاد تنازلي حيّ ودقيق لكل أجل قانوني أو موعد جلسة  
- تنبيهات متدرجة (قبل 7 أيام – 3 أيام – يوم – ساعتين – ساعة) مع إشعارات على الهاتف  
- ربط كل موعد برقم القضية، الخصم، المحكمة، نوع الإجراء، والوثائق المرفقة  
- مؤقت فوترة احترافي (Billing Timer) يحسب الوقت القابل للفوترة لكل جلسة أو مهمة  
- حفظ جميع البيانات محليًا في المتصفح بتشفير قوي (لا تُرسل إلى أي سيرفر)  
- بحث سريع وتصنيف حسب القضية، المحكمة، الحالة (منتهية – قيد التنفيذ – ملغاة)  
- تقويم مدمج مع عرض شهري وأسبوعي  
- وضع ليلي وتصميم متجاوب 100%  
- يعمل بدون إنترنت تمامًا بعد التثبيت الأول  
- قابل للتثبيت على الهاتف والكمبيوتر كتطبيق أصلي

#### التقنيات المستخدمة
- React + TypeScript  
- Vite (أسرع أداة بناء حاليًا)  
- Tailwind CSS  
- Zustand لإدارة الحالة  
- LocalForage + IndexedDB مع تشفير محلي  
- Workbox + Service Worker (PWA كامل المواصفات)  
- Push Notifications & Badging API  
- Manifest كامل + أيقونات مخصصة لكل الأجهزة

### English Version (Full README)

**Aajall – Legal Deadlines & Case Management PWA**  
The ultimate deadline tracker built exclusively for lawyers and law firms

#### Project Overview
Aajall (آجال) is a professional-grade Progressive Web App designed specifically for lawyers, law firms, and legal consultants to never miss a critical legal deadline, hearing, or filing date again.  
It features live countdown timers, tiered notifications, full case linking, billing timers, and 100% offline functionality with strong local encryption.

Developed by a Health Informatics Specialist with deep legal-tech experience and a Full-Stack Developer, Aajall combines mission-critical reliability with modern web performance.

#### Key Features
- Live countdown for every legal deadline and court hearing  
- Tiered smart notifications (7 days – 3 days – 1 day – 2 hours – 1 hour before deadline)  
- Link each event to case number, client, court, procedure type, and attached documents  
- Professional billing timer – track billable hours per session or task  
- All data stored locally with encryption – nothing ever leaves your device  
- Powerful search and filtering by case, court, status  
- Built-in calendar (monthly & weekly views)  
- Dark mode + fully responsive design  
- Works 100% offline after first visit  
- Installable on mobile and desktop as a native app

#### Tech Stack
- React + TypeScript  
- Vite  
- Tailwind CSS  
- Zustand (state management  
- LocalForage + IndexedDB with local encryption  
- Full PWA (Workbox, Service Workers, Push API, Badging)  
- Complete manifest and custom icons
